Chat Application

Skills Needed:

Node.js, Web Sockets, npm, JSON

Prattle is a chat application that I built during my time at General Assembly that runs in the browser. It currently supports multiple users, various slash commands, private messages, and a list of current users. I am planning on making it more robust by including standard features that are included in other chat application, including the use of chat rooms and a login system with a saved history.

Prattle was built using the web socket protocol. The client is built entirely in the browser, using JavaScript to handle outgoing and incoming messages and manipulating the DOM accordingly. Node.js was used to build the server. The communication between the server and client is facilitated using JSON objects. Each message sent on both ends contain types which tells the receiving agent what to do with it.
